利用規約

最終更新日：2025年1月20日

第1条（適用）

本利用規約（以下「本規約」）は、ミテルンデス（以下「本アプリ」）の利用に関する条件を定めるものです。ユーザーは、本アプリをダウンロード、インストール、または使用することにより、本規約に同意したものとみなされます。

第2条（定義）

本規約において使用する用語の定義は、以下のとおりとします。

  1. 「本アプリ」とは、ミテルンデス（Miterundesu）という名称のiOSアプリケーションを指します。
  2. 「ユーザー」とは、本アプリを利用するすべての個人を指します。
  3. 「運営者」とは、本アプリの開発・運営を行う個人または団体を指します。
  4. 「シアターモード」とは、映画館や美術館など静かな場所での使用を想定した、撮影機能を無効化するモードを指します。
  5. 「プレスモード」とは、報道関係者や開発者向けの、スクリーンショットや画面録画の制限を解除する特別モードを指します。

第3条（本アプリの目的と特性）

3.1 本アプリの目的

本アプリは、スマートフォンのカメラを拡大鏡として使用し、文字や物を見やすくすることを目的としています。撮影や録画を主目的とするものではありません。

3.2 本アプリの特性

本アプリには以下の特性があります：

  • 撮影した画像は10分後に自動的に削除されます
  • 画像の共有、保存、エクスポートはできません
  • スクリーンショットや画面録画は技術的に防止されています（プレスモードを除く）
  • すべてのデータは端末内にのみ保存され、外部に送信されません（プレスモード認証を除く）
ご注意：本アプリは「見るため」のツールであり、証拠保全や記録保存を目的とした使用には適していません。重要な情報は別の方法で記録してください。

第4条（利用条件）

4.1 年齢制限

本アプリは13歳以上の方を対象としています。13歳未満の方は保護者の同意を得た上でご利用ください。

4.2 デバイス要件

本アプリを使用するには、以下の要件を満たすiOSデバイスが必要です：

  • iOS 17.0以降
  • カメラ機能を備えたiPhone
  • 十分なストレージ空き容量

4.3 カメラ権限

本アプリの使用には、カメラへのアクセス許可が必要です。権限を拒否した場合、アプリの機能を利用できません。

第5条（禁止事項）

ユーザーは、本アプリの利用にあたり、以下の行為を行ってはなりません。

5.1 法令違反

  • 日本国内外の法令、条例、規則に違反する行為
  • 盗撮、ストーカー行為、その他の犯罪行為
  • 個人のプライバシーを侵害する行為

5.2 施設のルール違反

  • 撮影禁止の場所での使用（本アプリが拡大鏡目的であっても、施設のルールを優先してください）
  • 美術館、博物館、映画館等の施設が定める利用規則に反する行為

5.3 技術的な不正行為

  • 本アプリのセキュリティ機能を回避、無効化、または改ざんする行為
  • 本アプリのソースコードをリバースエンジニアリング、逆コンパイル、または逆アセンブルする行為
  • 本アプリの機能を不正に改変または複製する行為
  • プレスモードを不正に取得または使用する行為

5.4 商業利用

  • 本アプリを商業目的で使用する行為（プレスモードによる報道利用を除く）
  • 本アプリを用いた監視ビジネスやサービスの提供

5.5 その他の迷惑行為

  • 他のユーザーや第三者に迷惑をかける行為
  • 本アプリの運営を妨害する行為
  • 本アプリの信用を毀損する行為

第6条（プレスモード）

6.1 プレスモードの目的

プレスモードは、以下の目的のために提供されます：

  • 報道機関による取材活動
  • アプリ開発者による開発・テスト作業
  • 運営者が認めたその他の正当な目的

6.2 プレスモードの申請

プレスモードの利用を希望するユーザーは、以下の情報を提供して申請する必要があります：

  • デバイスID（アプリ内で表示されるUUID）
  • 所属組織名
  • 連絡先（メールアドレスまたは氏名）
  • 利用目的

6.3 プレスモードの承認

運営者は、申請内容を審査し、承認の可否を決定します。運営者は理由を開示することなく申請を拒否することができます。

6.4 プレスモードの有効期限

承認されたプレスモードには有効期限が設定されます。有効期限を過ぎると、プレスモードは自動的に無効化されます。

6.5 プレスモードの取り消し

運営者は、以下の場合にプレスモードを即座に取り消すことができます：

  • ユーザーが本規約に違反した場合
  • 不正な目的での使用が判明した場合
  • その他、運営者が不適切と判断した場合
重要：プレスモードを不正に使用した場合、法的措置を取る可能性があります。

第7条（知的財産権）

7.1 本アプリの知的財産権

本アプリに関する著作権、商標権、特許権、その他の知的財産権は、すべて運営者または正当な権利者に帰属します。

7.2 撮影画像の権利

ユーザーが本アプリで撮影した画像の著作権は、ユーザーに帰属します。ただし、以下の点にご注意ください：

  • 画像は10分後に自動的に削除されます
  • 画像には透かし（タイムスタンプとデバイスID）が埋め込まれます
  • 運営者は、画像データにアクセスすることはできません

7.3 ライセンス

ユーザーは、本アプリを個人的かつ非商業的な目的でのみ使用するライセンスを付与されます。このライセンスは、譲渡不可能かつ非独占的です。

第8条（免責事項）

8.1 サービスの提供

本アプリは「現状有姿」で提供されます。運営者は、以下について保証しません：

  • 本アプリの機能が中断なく動作すること
  • 本アプリにエラーやバグが含まれないこと
  • 本アプリが特定の目的に適合すること

8.2 セキュリティ機能

本アプリはスクリーンショットや画面録画を防止する技術的措置を講じていますが、すべての状況において完全に防止できることを保証するものではありません。

8.3 データの保存

撮影画像は10分後に自動削除されます。この削除は復元不可能です。重要な情報は別の方法で記録してください。

8.4 施設のルール

本アプリを使用する際は、使用場所の規則やルールを遵守してください。本アプリの使用が原因で施設から退場を求められた場合でも、運営者は一切責任を負いません。

8.5 損害賠償の制限

運営者は、本アプリの使用または使用不能により生じたいかなる損害（直接損害、間接損害、逸失利益を含む）についても責任を負いません。ただし、運営者の故意または重過失による場合を除きます。

第9条（アプリの変更・中止）

9.1 機能の変更

運営者は、ユーザーへの事前通知なく、本アプリの機能を追加、変更、または削除することができます。

9.2 サービスの中止

運営者は、以下の場合に本アプリの提供を中止することができます：

  • システムメンテナンスや緊急の保守作業が必要な場合
  • 天災、事変その他の非常事態が発生した場合
  • その他、運営上やむを得ない事情がある場合

9.3 サービスの終了

運営者は、1ヶ月前までにユーザーに通知することにより、本アプリの提供を終了することができます。

第10条（利用規約の変更）

運営者は、必要に応じて本規約を変更することができます。変更後の規約は、本ページおよびアプリ内で通知します。変更後も本アプリを継続して使用する場合、変更後の規約に同意したものとみなされます。

第11条（準拠法および管轄裁判所）

11.1 準拠法

本規約は、日本法に準拠して解釈されます。

11.2 管轄裁判所

本規約に関する紛争については、東京地方裁判所を第一審の専属的合意管轄裁判所とします。

第12条（分離可能性）

本規約のいずれかの条項が無効または執行不能と判断された場合でも、その他の条項は引き続き有効です。
