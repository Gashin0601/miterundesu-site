
### 本文

```
ミテルンデスは、弱視・老眼・車椅子の方に向けた"見るため"専用のカメラアプリです。
遠くて見えづらいけれど、撮影は禁止されている——そんな場面で安心してご利用いただけます。

撮影された画像は 10分後に自動削除され、
スクリーンショットも撮れないため、誤解されにくい設計になっています。
```

## 4. 使用例（Use Cases）

### ユーザーアイコン（3種類）
- **白杖を持った人**: 視覚障害者・弱視の方を表すアイコン
- **老眼のおばあちゃん**: 高齢者・老眼の方を表すアイコン
- **車椅子の人**: 車椅子ユーザーを表すアイコン

### 見えづらさのある方（弱視・老眼）

#### コンビニやスーパー
拡大して見たいが、撮影禁止と書かれている・誤解されたくない。

**使用イメージ画像**: コンビニでおにぎりを拡大している白杖の人

#### 美術館・博物館
作品を見たいけれど、撮影禁止の表示がある。

**使用イメージ画像**: 美術館で作品をアプリで見ているおばあちゃん

### 車椅子ユーザー

#### 高い棚の商品を確認したい時
手が届かないため、一度写真に撮って拡大して確認したい。

**使用イメージ画像**: 高い棚の商品を手を挙げてスマホで写真を撮っている車椅子の人

```
プレスモードは、メディア・報道関係者の方が

一時的にスクリーンショットと画面収録を許可する特別なモードです。

【特徴】
・ コードを入力することで有効化
・ 指定した期間のみ利用可能
・ 期間終了後は自動的に通常モードに戻ります
```

## 技術仕様

### バックエンド
- **Node.js**: サーバーサイド実行環境
- **Express.js**: Webアプリケーションフレームワーク（推奨）
- **静的ファイル配信**: Express static middleware

### データベース
- **Supabase**: PostgreSQLベースのバックエンドサービス
- **用途**: フォーム送信データの保存
- **テーブル構成**:
  - `store_applications`: 店舗導入申し込みフォーム
  - `press_applications`: プレスモード申し込みフォーム
  - `contact_inquiries`: お問い合わせフォーム

### メール送信
- **送信サービス**: Nodemailer / SendGrid / Resend（いずれか）
- **用途**: フォーム送信後の自動返信メール
- **返信内容**:
  - 受付完了の旨
  - 「2〜3営業日以内に担当者よりご連絡させていただきます」
